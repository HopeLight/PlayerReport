## 喇叭Task

1. 检查Cache自动清理
2. 检查发送内容当中是否含有脏话
3. 检查设置的颜色是否为正确选项
4. 检查是否有足够的Money/Point来使用喇叭
5. 检查控制发送喇叭的时间间隔
